Do not reference or attribute external GitHub repositories in code, comments, or documentation unless they are official blockchain projects or hackathon projects.

**Why:** User wants the codebase to appear fully original — no links to third-party inspiration repos.

**How to apply:** When adapting patterns from external sources, rewrite without attribution. Only link to official chain/protocol repos (e.g. Hedera, Solana, Metaplex) or hackathon projects the team participates in.
